# 盐选付费节点研究

## 付费墙位置

- 付费卡点在 **6000-6500 字**左右截断
- 对于 1.5 万字短篇，约在 **40%** 位置
- 对于 8000 字短篇，约在 **75-80%** 位置
- 编辑评估标准：读者在卡点位置是否有付费冲动

## 节奏要求

- 前 500 字必须亮出冲突与悬念
- 2000 字内埋 3 个钩子
- 每 2000 字至少一个情节转折
- 免费章节 ≤ 6000 字，在关键反转前设付费点

## 对结构的影响

- 40% 的位置前读者有完整的情绪铺垫空间
- 付费墙应卡在情绪最高点、共鸣最强的节点——让读者带着"然后呢"的冲动付费
- 不能在某一段长时间平铺，后半段也需要持续的小转折维持张力
- 真正的大反转应在付费之后，作为付费读者的奖励
